# Dog AR Project

iPhone 14 ProのLiDARスキャナと3Dスキャンアプリを使って、愛犬を3Dモデル化し、Web上でAR表示できるようにしたプロジェクトです。

## GitHub Pages URL

`https://yvrwwq9bgt-ai.github.io/Repository-name-dog-ar-project-/`

## Included Model

このZIPには、アップロードされた3Dモデルを `models/dog.glb` として入れてあります。

## Features

- 愛犬の3DモデルをWeb上で表示
- スマートフォンからAR表示に対応
- GLB形式の3Dモデルを使用
- GitHub Pagesで公開可能

## Tools Used

- iPhone 14 Pro
- LiDAR Scanner
- Blender
- GitHub Pages
- HTML / CSS / JavaScript
- model-viewer
